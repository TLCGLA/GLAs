Avoiding Run-on Sentences
Introduction

This Guided Learning Activity will help you to improve your sentences by identifying key sentence components and learning how to connect them.

Student Learning Outcomes

By the end of this activity, you should be able to:

  • Identify the components of an independent clause
  • Correct run-on errors between independent clauses using periods, semi-colons, and coordinating conjunctions
