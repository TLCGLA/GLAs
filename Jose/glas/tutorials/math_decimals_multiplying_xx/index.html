Multiplication and Division of Decimals
Introduction

Decimals are everywhere and knowing how to handle them is a valuable skill in life. In this learning activity, you will learn how to multiply and divide decimals.

Student Learning Outcomes

By the end of this activity, you should be able to:

  • Identify and name decimal place values
  • Round decimals to a given place value
  • Multiply decimals
  • Divide decimals
