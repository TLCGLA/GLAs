MobilySlider

Posted by: Marcin Dziewulski on 20.01.11

MobilySlider is advanced slideshow plugin for jQuery (5KB).

  • can add inline HTML content (awesome!)
  • looping
  • auto generated pagination
  • auto generated arrows
  • autoplay
  • fade, horizontal, vertical slide transition effect
  • pause on hover
  • callback functions

Changelog:

  1. 03.10.2011 - fixed animation bug in FF6 and Chrome
  2. 02.02.2011 - added fade transition

Featured on:

Speckyboy

JavaScript:

  1. $('.slider').mobilyslider({
  2. 	content: '.sliderContent', // class for slides container
  3. 	children: 'div', // selector for children elements
  4. 	transition: 'horizontal', // transition: horizontal, vertical, fade
  5. 	animationSpeed: 300, // slide transition speed (miliseconds)
  6. 	autoplay: false, 
  7. 	autoplaySpeed: 3000, // time between transitions (miliseconds)
  8. 	pauseOnHover: false, // stop animation while hovering
  9. 	bullets: true, // generate pagination (true/false, class: sliderBullets)
  10. 	arrows: true, // generate next and previous arrow (true/false, class: sliderArrows)
  11. 	arrowsHide: true, // show arrows only on hover
  12. 	prev: 'prev', // class name for previous button
  13. 	next: 'next', // class name for next button
  14. 	animationStart: function(){}, // call the function on start transition
  15. 	animationComplete: function(){} // call the function when transition completed
  16. });

HTML:

  1. <div class="slider">
  2. 	<div class="sliderContent">
  3. 		<div class="item">
  4. 			<!-- any html content! -->
  5. 		</div>
  6. 		<!-- next slides -->
  7. 	</div>
  8. </div>
