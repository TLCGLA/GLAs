Object  Pronouns
•When a noun is an object replace it with an object pronoun.
•
•Sample:
•Peter ate an apple. He liked it so much that he asked for one more, but his mother gave him two.
•
•“It”  refers to “an apple,” which is in the object position in the first sentence.
•“Him” refers to “Peter” which is in the object position in the second sentence.
•Instead of saying “he liked an apple,” we say “he liked it,” and instead of saying “his mother gave Peter” we say “his mother gave him.”
•
•Some object pronouns are:  me, you, him, her, it, us, them, etc.
