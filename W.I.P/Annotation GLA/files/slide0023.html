How to Outline a Textbook
uMost textbooks are already written using an outline format, so creating an outline using a textbook is usually fairly easy once you know how.
uTry using the following system:
uChapter title:  Outline Title
uSection Headers:  Roman Numerals
uSubheaders:  Capitalized Letters
uSub-subheaders:  Arabic Numerals
uTopic Sentences:  Lower-case letters
Donatelle, Rebecca.  Health:  The Basics.  10th Edition.  Boston:  Pearson, 2013.  Print.
