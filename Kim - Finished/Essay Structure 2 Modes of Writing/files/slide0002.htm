Agenda
 This workshop will cover the following modes:
1.Definition
2.Description
3.Exposition
4.Exemplification
5.Classification
6.Compare / Contrast
7.Process
8.Persuasion
9.Cause/Effect
10.Problem / Solution
11.Narrative
 
Note:  Each mode of writing includes a sample using the topic of the American Dream.  You can use just about any mode of writing for any topic.  It is up to you to decide which mode works best for the assignment you’re working on.
