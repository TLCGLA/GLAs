USE A COMMA FOR INTRODUCTORY CLAUSES
•Use a comma after an introductory clause.  An introductory clause is a dependent clause that provide background information for the rest of the sentence.
•Introductory clauses start with adverbs like: after, although, as, because, before, if, since, though, until, when, etc.
•Examples: 
•Before going to The Learning Center, you must get a student ID card.
•If you are a vegetarian, you can make the sauce without meat.
•Note:  If the dependent clause comes at the end of the sentence, you don’t need a comma.
•Example:  You must getting a student ID card before going to The Learning Center.
