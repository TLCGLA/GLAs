Note the difference !
Independent clause:
Powerpoints are so awesome.

Dependent clause:
because powerpoints are so awesome 

Note how the independent clause is basically a simple sentence: Powerpoints are so awesome. It has a subject, “Powerpoints,” and a verb, “are.”  It also expresses a complete thought.  In contrast, “because powerpoints are so awesome” also obviously has a subject and a verb, but does not express a complete thought.
