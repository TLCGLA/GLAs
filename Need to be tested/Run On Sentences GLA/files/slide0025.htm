Strategy 3
•Separate the independent clauses with a comma AND a coordinating conjunction.
•The coordinating conjunctions are:  for, and, nor, but, or, yet, so.  You can remember them with the acronym FANBOYS.
•
•Incorrect: Oscar wants to write screenplays for a living, he is double majoring in film and English.
•
•Correct: Oscar wants to write screenplays for a living, so he is double majoring in film and English.
