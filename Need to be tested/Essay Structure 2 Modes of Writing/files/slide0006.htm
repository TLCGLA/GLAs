Extended Definition
 Write an extended definition (one paragraph or more) for complex ideas.  Plan to write an extended definition in the following circumstances:
◦Understanding the term is vital to understanding the entire piece of writing. 
◦There is already some confusion or controversy about the words definition.
◦The term is so difficult to understand that it requires a detailed definition.
 
 To write an extended definition, begin with the term = type + distinguishing characteristics format.  Then, include any supplemental information the reader needs to understand the term.
