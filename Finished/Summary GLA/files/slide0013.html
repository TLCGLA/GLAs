Step 3:  Find the Main Points
¨Once you have read the text carefully, review any notes you have made.  Then, develop a list of main points and key examples that you could use to explain the article.
¨
¨Main points are the pieces of information readers need to understand about the article.
¨Key examples help show what the main points mean.  Although summaries don’t use many details, you may want to include one or two key examples.
