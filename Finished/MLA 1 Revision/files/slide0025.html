USING A HANDBOOK TO CREATE A WORKS CITED PAGE
•A handbook like Rules for Writers is a valuable tool in creating a properly-formatted Works Cited page.
•You should be able to find the MLA section of your handbook quickly.  Often, the margins of the pages are shaded to help you find them.
•The first step to creating a proper citation is to know what type of source you’re using (book, article, website, etc.).
•Find that source type in your handbook using the table of contents in the MLA section.
•The handbook will guide you through the proper format for citing that type of source.
