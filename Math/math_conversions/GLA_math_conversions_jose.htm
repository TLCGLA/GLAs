Conversions and Dimensional Analysis


Introduction


Welcome to the Conversions and Dimensional Analysis guided learning activity. By the end of this activity, you will be able to set up and solve various kinds of dimensional analysis problems.

By the end of this activity, you should be able to:
  • perform unit conversion using dimensional analysis
  • report answers with correct number of significant figures in scientific notation


Let's get started!


If at any time you need help, please ask a tutor.

Step 1:

First you'll need to make sure you have the handouts for this activity. If you don't, ask a tutor to help you.

Step 2:

First, you need to open a PowerPoint on dimensional analysis.

Step 3:

Once the PowerPoint is open, read through slides #1-10. Pay particular attention to how the example problems are set up and common units used.

Step 4:

On a separate piece of paper, work through the two practice problems given in slides #11-12. Make sure to set up the problems according to the directions on slide #9. Ask a tutor for help if you get stuck.

STOP and review your answers with a tutor.

Step 5:

Read through slides #13-15. Pay particular attention to how the example problems are set up.

Step 6:

On a separate piece of paper, work through the two practice problems given in slides #16-17. Make sure to set up the problems according to the directions on slide #13. Ask a tutor for help if you get stuck.

STOP and review your answers with a tutor.

Step 7:

Read through slides #18-20. Pay particular attention to how the example problems are set up.

Step 8:

On a separate piece of paper, work through the practice problem given in slide #21. Make sure to set up the problems according to the directions on slide #20. Ask a tutor for help if you get stuck.

STOP and review your answers with a tutor.

Step 9:

Let’s see how much you have learned by completing this quiz. Use a blank sheet of paper for your work.

Step 10:

Review your work with a tutor.

That's it. You are done! Thank you for completing this guided learning activity.
