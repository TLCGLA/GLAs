APA 1 (Formatting and Structure)
Introduction

Many of the papers you have to write in college must follow a set of format guidelines. In the humanities, it is common to use the Modern Language Association style (MLA), while in the sciences, papers follow the American Psychological Association style (APA).

In this guided learning activity, you will cover an introduction the the APA style.

Student Learning Outcomes

By the end of this activity, you should be able to:

  • Identify APA 6th Edition guidelines, including formatting, and structure of an APA paper.
